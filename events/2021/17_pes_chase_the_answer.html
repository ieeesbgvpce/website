Introduction

Greetings everyone!
We members of IEEE SB PES GVPCE (A) has now come up with an exciting and challenging event,CHASE THE ANSWER, the virtual treasure hunt.

GUIDELINES :

👉🏻Follow IEEE's instagram account and look out for the stories posted on the event dates.
👉🏻Each story has a question and a answer belonging to some other question.
👉🏻To be more clear,the answer to the first question is placed in a story along with the following question and so on.
👉🏻The answer for every question has an attached binary number to it i.e (1 or 0).
👉🏻Note down the binary number and create a binary code in a sequential manner which will be your final answer.
👉🏻That code must be entered in the last story.
👉🏻The player who gets the right code quickly will be the winner.
Note:- The stories are jumbled (You can't find the clues and their answers one after the other.)

📌 NOTE:

No Registration fee.

Certificate of appreciation will be provided to top 3 players.

Certificate of participation will be provided to other players.
